What is a Recursion ?
When a function calls itself until a specified condition is met

Example - Infinite Recursion


def pyout(n):
    print(n)

    pyout(n+1)

n = 1
pyout(n)

RESULT - Goes into Infinite loop and gives segmentation fault/stack overflow

SOLUTION - You will have to mention the base condition (when the process should be stop)

------------------------------------------------------------------------------------------------------
# BASE CASE OR BASE CONDITION:
 - The condition where the function stops calling itself and returns the value, this prevents the INFINITE loop


 Example - using Base condition

def pyout(n):

    if n == 27+1:   # Using Base Condition , it prevents the infinite loop
        return

    print(n)

    pyout(n+1)

n = 1
pyout(n)
